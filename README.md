# py-rap-gen

Rap lyric generatinng system for python.

## Usage

Run following commnads under the repository directory.

```sh
pip install poetry
poetry install
poetry run preprocess
```

Run CLI application.

```sh
petry run generate
```

Run WEB application.

```sh
petry run demo
```

Sample Site.
http://34.97.214.0:8080
